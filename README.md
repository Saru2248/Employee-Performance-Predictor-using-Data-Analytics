
# 🏢 Employee Performance Predictor using Data Analytics

## 🚀 Overview
This project develops a Machine Learning system to **predict employee performance** using HR-related data.

The objective is to help organizations transition from **subjective evaluations** to **data-driven decision-making**.

The model classifies employees into:
- 🟢 High Performer  
- 🟡 Medium Performer  
- 🔴 Low Performer  

---

## 🎯 Problem Statement
In many organizations:
- Performance reviews are subjective and inconsistent  
- Promotions may involve bias  
- Training resources are not effectively utilized  
- Low performers are identified too late  

This project addresses these challenges using **data analytics and machine learning techniques** to predict employee performance accurately.

---

## 💼 Business Value

- Identify high performers for promotions and rewards  
- Detect low performers early for improvement plans  
- Optimize training and development programs  
- Enable data-driven HR decision-making  

---

## ⚙️ Tech Stack

- **Programming Language:** Python  
- **Data Processing:** Pandas, NumPy  
- **Visualization:** Matplotlib, Seaborn  
- **Machine Learning:** Scikit-learn  
- **Model Used:** Random Forest Classifier  
- **Dataset:** Synthetic HR dataset (simulating real-world scenarios)

## 🚀 How to Run the Project

### 1. Clone the Repository
```bash
git clone https://github.com/Saru2248/Employee-Performance-Predictor.git
cd Employee-Performance-Predictor
````

### 2. Create Virtual Environment

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Mac/Linux
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Project

```bash
python main.py
```
